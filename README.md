# PORTIFOLIO-JULIA

Este portifólio foi construído como um dos intregáveis da disciplina de Introdução a Computação, nele eu apresento de maneira geral os aspectos mais importantes da minha tragetória acadêmica, como minha hard e soft skills e os projetos venho desdenvolvendo durante a minha formação. 

Para realizar este projeto, fiz o uso de HTML, CSS, Java Script e do versel.

O projeto têm 3 páginas:
-> Inicial: Na qual eu me apresento de maneira breve e mostro as formas de contato.
-> Skills: Na 2 estão as minhas hard (na primeira coluna) e soft(na segunda coluna) skills, usei a criatividade e pus as habilidades em uma 
"lista de compras de habilidades" , com todas marcadas cono comprada , ao lado , um carrinho cheio de sacolar (indicando as habilidades compradas) e um pouco acima há uma placa escrito "nunca pare de comprar" , ou seja, nunca pare de adquirir novas habilidades.
-> Projetos: Na 3 estão meus projetos , o DoniNod (jogo que estou desenvolvendo na disciplina de projetos 1), o projeto de python(desenvolvimento de um programa funcional em python para acompanhar o treinamento do usuário para um HYROX), O Chamado da Ilha Viva(um jogo No/Code inspirado no filme "Moana") e um artigo científico que publiquei na época em que fazia medicina da UPE.

Tentei fazer este projeto de uma forma que comunicasse tudo isso, de uma forma criativa , que trasmitisse a minha essência, e particularmente estou muito feliz com o resultado!
